PugSharp

Pugsharp is a PUG System Plugin for CS2 based on the awsome CounterStrikeSharp by roflmuffin.

Its intended purpose is to be used with our fork of eventula, but ofc can be used in a different environment or with different software as well. If you implement software to interface with PugSharp let us know please!

Warning

This Plugin is in a very early state of development and some things are not fully working right now! Please report any issues you find either on Discord or in our issues tab on Github

Admins quickstart note

  • If you want to deploy PugSharp, please take a look into the Admins quickstart guide

  • If you want to help with the translation of PugSharp, please check out our Translation / i18n section

Developer note

  • Github url: https://github.com/Lan2Play/PugSharp

  • If you want to contribute to PugSharp, please take a look into the Developers start guide
